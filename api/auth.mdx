## POST /auth/login

Authenticate a user and receive a JWT token.

### Request

<ParamField body="email" type="string" required>
  User's email address
</ParamField>

<ParamField body="password" type="string" required>
  User's password
</ParamField>

### Request Example

```json
{
  "email": "[email protected]",
  "password": "SecurePassword123"
}
```

### Response

<ResponseField name="token" type="string" required>
  JWT authentication token to use for subsequent requests
</ResponseField>

### Response Example

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJ1c2VyXzEyMyIsImVtYWlsIjoiam9obkBleGFtcGxlLmNvbSIsImlhdCI6MTcwOTEyMzQ1NiwiZXhwIjoxNzA5MjA5ODU2fQ.abc123def456ghi789"
}
```

### Error Responses

<Expandable title="400 Bad Request">
```json
{
  "error": "Invalid email or password",
  "code": "INVALID_CREDENTIALS",
  "statusCode": 400
}
```
</Expandable>

<Expandable title="401 Unauthorized">
```json
{
  "error": "Account is locked or disabled",
  "code": "ACCOUNT_LOCKED",
  "statusCode": 401
}
```
</Expandable>

### Implementation Example

```dart
static Future<String?> login(String email, String password) async {
  final response = await ApiService.dio.post(
    ApiRoutes.authLogin,
    data: {'email': email, 'password': password},
  );
  final token = response.data['token'] as String?;
  if (token != null) {
    await _saveToken(token);
    ApiService.setAuthToken(token);
  }
  return token;
}
```

## POST /auth/register

Register a new user account and receive a JWT token.

### Request

<ParamField body="firstName" type="string" required>
  User's first name
</ParamField>

<ParamField body="lastName" type="string" required>
  User's last name
</ParamField>

<ParamField body="email" type="string" required>
  User's email address (must be unique)
</ParamField>

<ParamField body="password" type="string" required>
  User's password (must meet minimum security requirements)
</ParamField>

### Request Example

```json
{
  "firstName": "John",
  "lastName": "Doe",
  "email": "[email protected]",
  "password": "SecurePassword123"
}
```

### Response

<ResponseField name="token" type="string" required>
  JWT authentication token for the newly created user
</ResponseField>

### Response Example

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJ1c2VyXzEyNCIsImVtYWlsIjoiam9obkBleGFtcGxlLmNvbSIsImlhdCI6MTcwOTEyMzQ1NiwiZXhwIjoxNzA5MjA5ODU2fQ.xyz789abc123def456"
}
```

### Error Responses

<Expandable title="400 Bad Request - Email Already Exists">
```json
{
  "error": "Email address already registered",
  "code": "EMAIL_EXISTS",
  "statusCode": 400
}
```
</Expandable>

<Expandable title="400 Bad Request - Weak Password">
```json
{
  "error": "Password does not meet security requirements",
  "code": "WEAK_PASSWORD",
  "statusCode": 400
}
```
</Expandable>

<Expandable title="400 Bad Request - Invalid Email">
```json
{
  "error": "Invalid email format",
  "code": "INVALID_EMAIL",
  "statusCode": 400
}
```
</Expandable>

### Implementation Example

```dart
static Future<String?> register(
  String firstName,
  String lastName,
  String email,
  String password,
) async {
  final response = await ApiService.dio.post(
    ApiRoutes.authRegister,
    data: {
      'firstName': firstName,
      'lastName': lastName,
      'email': email,
      'password': password,
    },
  );
  final token = response.data['token'] as String?;
  if (token != null) {
    await _saveToken(token);
    ApiService.setAuthToken(token);
  }
  return token;
}
```

## POST /auth/logout

Invalidate the current user session and clear authentication.

<Note>
This endpoint requires authentication via the `Authorization: Bearer {token}` header.
</Note>

### Request

No request body required. The token is taken from the `Authorization` header.

### Response

```json
{
  "message": "Logged out successfully"
}
```

### Error Responses

<Expandable title="401 Unauthorized">
```json
{
  "error": "Invalid or expired token",
  "code": "UNAUTHORIZED",
  "statusCode": 401
}
```
</Expandable>

### Implementation Example

```dart
static Future<void> logout() async {
  final prefs = await SharedPreferences.getInstance();
  await prefs.remove(_tokenKey);
  ApiService.clearAuthToken();
}
```

<Info>
The client implementation clears the local token regardless of server response. This ensures logout works even if the server is unreachable.
</Info>

## GET /auth/me

Retrieve the authenticated user's profile information.

<Note>
This endpoint requires authentication via the `Authorization: Bearer {token}` header.
</Note>

### Request

No request body required. User is identified from the JWT token.

### Response

<ResponseField name="id" type="string" required>
  Unique user identifier
</ResponseField>

<ResponseField name="firstName" type="string" required>
  User's first name
</ResponseField>

<ResponseField name="lastName" type="string" required>
  User's last name
</ResponseField>

<ResponseField name="email" type="string" required>
  User's email address
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of account creation
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last profile update
</ResponseField>

### Response Example

```json
{
  "id": "user_123",
  "firstName": "John",
  "lastName": "Doe",
  "email": "[email protected]",
  "createdAt": "2024-01-15T10:30:00Z",
  "updatedAt": "2024-02-20T14:45:00Z"
}
```

### Error Responses

<Expandable title="401 Unauthorized">
```json
{
  "error": "Invalid or expired token",
  "code": "UNAUTHORIZED",
  "statusCode": 401
}
```
</Expandable>

<Expandable title="404 Not Found">
```json
{
  "error": "User not found",
  "code": "USER_NOT_FOUND",
  "statusCode": 404
}
```
</Expandable>
