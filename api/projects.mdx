## Overview

Projects belong to workspaces and contain SRS documents, requirements, and related artifacts. Users access projects through their workspace membership.

<Note>
All project endpoints require authentication via the `Authorization: Bearer {token}` header.
</Note>

---

## GET /projects

Retrieve all projects the authenticated user has access to across all workspaces.

### Request

No request parameters required. Returns projects based on user's workspace memberships.

### Response

Returns an array of project objects.

<ResponseField name="id" type="string" required>
  Unique project identifier
</ResponseField>

<ResponseField name="name" type="string" required>
  Project name
</ResponseField>

<ResponseField name="description" type="string">
  Project description (optional)
</ResponseField>

<ResponseField name="workspaceId" type="string" required>
  Identifier of the workspace this project belongs to
</ResponseField>

<ResponseField name="workspaceName" type="string" required>
  Name of the workspace this project belongs to
</ResponseField>

<ResponseField name="documentCount" type="integer" required>
  Number of SRS documents in this project
</ResponseField>

<ResponseField name="requirementCount" type="integer" required>
  Total number of requirements across all documents
</ResponseField>

<ResponseField name="status" type="string" required>
  Project status: `active`, `archived`, or `completed`
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of project creation
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last project update
</ResponseField>

## GET /projects/:id

Retrieve detailed information about a specific project.

### Path Parameters

<ParamField path="id" type="string" required>
  Project identifier
</ParamField>

### Request Example

```
GET /projects/proj_1
```

### Response

<ResponseField name="id" type="string" required>
  Unique project identifier
</ResponseField>

<ResponseField name="name" type="string" required>
  Project name
</ResponseField>

<ResponseField name="description" type="string">
  Project description (optional)
</ResponseField>

<ResponseField name="workspaceId" type="string" required>
  Identifier of the workspace this project belongs to
</ResponseField>

<ResponseField name="workspaceName" type="string" required>
  Name of the workspace this project belongs to
</ResponseField>

<ResponseField name="status" type="string" required>
  Project status: `active`, `archived`, or `completed`
</ResponseField>

<ResponseField name="documents" type="array" required>
  Array of SRS documents in this project
  
  <Expandable title="Document Object Structure">
    <ResponseField name="id" type="string" required>
      Document identifier
    </ResponseField>
    
    <ResponseField name="title" type="string" required>
      Document title
    </ResponseField>
    
    <ResponseField name="version" type="string" required>
      Current document version (e.g., "1.0.0")
    </ResponseField>
    
    <ResponseField name="status" type="string" required>
      Document status: `draft`, `review`, `approved`, or `published`
    </ResponseField>
    
    <ResponseField name="requirementCount" type="integer" required>
      Number of requirements in this document
    </ResponseField>
    
    <ResponseField name="lastModifiedBy" type="string" required>
      Name of user who last modified the document
    </ResponseField>
    
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp of document creation
    </ResponseField>
    
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of last document update
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="collaborators" type="array" required>
  Array of users who have contributed to this project
  
  <Expandable title="Collaborator Object Structure">
    <ResponseField name="id" type="string" required>
      User identifier
    </ResponseField>
    
    <ResponseField name="firstName" type="string" required>
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string" required>
      User's last name
    </ResponseField>
    
    <ResponseField name="email" type="string" required>
      User's email address
    </ResponseField>
    
    <ResponseField name="lastActivity" type="string" required>
      ISO 8601 timestamp of user's last activity in this project
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdBy" type="object" required>
  Information about the user who created the project
  
  <Expandable title="Creator Object Structure">
    <ResponseField name="id" type="string" required>
      User identifier
    </ResponseField>
    
    <ResponseField name="firstName" type="string" required>
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string" required>
      User's last name
    </ResponseField>
    
    <ResponseField name="email" type="string" required>
      User's email address
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of project creation
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last project update
</ResponseField>
