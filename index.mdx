## What is FSD Movil?

FSD Movil is a comprehensive mobile platform built with Flutter that revolutionizes how teams manage software projects and generate technical documentation. It combines workspace collaboration, project tracking, and automated SRS (Software Requirements Specification) document generation in a single, cross-platform application.

## Platform Support

FSD Movil runs natively on all major platforms:

- **Mobile**: Android and iOS
- **Desktop**: Windows, macOS, and Linux
- **Web**: Browser-based access
